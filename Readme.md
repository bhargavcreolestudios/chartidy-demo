# Chartidy

Display the chart of table on user choice.

## Local Setup

1. Clone the repo
2. Go to Chrome extention page
3. Enable developer mode
4. Click on Load unpacked
5. Select the repo (chartidy-demo) 

## Library

1. Jquery
2. CanvasJS (Charts)
